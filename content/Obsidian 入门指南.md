# Obsidian 入门指南

Obsidian 是一款强大的本地笔记和知识管理工具，基于 Markdown 文件存储，让你的笔记完全可控。

## 为什么选择 Obsidian？

- **本地存储** - 笔记保存在本地，不用担心云服务涨价或服务关闭
- **双向链接** - [[双向链接]] 可以帮助建立知识网络
- **插件生态** - 丰富的社区插件，可以无限扩展功能
- **免费使用** - 个人使用完全免费

## 基础操作

### 创建笔记

1. 按 `Ctrl + N` (Windows) 或 `Cmd + N` (Mac) 创建新笔记
2. 开始输入内容，使用 Markdown 语法

### 链接笔记

使用 `[[笔记名称]]` 语法可以创建笔记之间的链接。

### 使用标签

在笔记开头添加 `tags` 字段来分类：

```yaml
---
tags:
  - 笔记
  - 教程
---
```

## 推荐插件

| 插件名称 | 功能 |
|---------|------|
| Templater | 高级模板功能 |
| Dataview | 数据查询和索引 |
| Excalidraw | 白板绘图 |

## 进阶技巧

> 知识不是线性积累的，而是通过网络连接的方式增长的。

1. **建立知识图谱** - 利用 Obsidian 的图谱视图查看笔记之间的联系
2. **每日笔记** - 建立晨间写作习惯
3. **定期回顾** - 用 Zettelkasten 方法整理知识
